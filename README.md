# IOT-BUOY
IoT-Based Coral Reef Environmental Monitoring &amp; Analytics System

Developed an end-to-end environmental monitoring and decision-support system for coral reef conservation by integrating IoT-based data collection, cloud data processing, and interactive analytics. Contributed to sensor data processing, ETL pipeline development, trend analysis, threshold-based alerts, and dashboard visualization for monitoring environmental factors such as sea temperature, turbidity, salinity, and light intensity. The solution utilized solar-powered IoT buoy devices, MQTT communication, Firebase cloud storage, and real-time analytics to support data-driven coastal management decisions.
